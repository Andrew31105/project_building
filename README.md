Role: Backend Developer
Tech stack: Java, Java Servlet/JSP, JDBC, MySQL, HTML/CSS/JS, MVC Pattern

Description:
Xây dựng hệ thống quản lý tòa nhà bao gồm các chức năng cơ bản cho phép quản trị viên quản lý thông tin tòa nhà, danh sách nhân viên, khách thuê và các dịch vụ liên quan. Ứng dụng được phát triển theo mô hình MVC, sử dụng JSP/Servlet làm backend, kết nối MySQL database thông qua JDBC và giao diện được xây dựng bằng HTML/CSS/JS thuần.

Key Features:

Quản lý thông tin tòa nhà: tên tòa, địa chỉ, diện tích, mô tả.

CRUD cho phòng ban, nhân viên, khách thuê.

Chức năng tìm kiếm, lọc dữ liệu theo nhiều tiêu chí.

Phân trang dữ liệu để tối ưu hiệu năng hiển thị.

Kiểm tra & validate dữ liệu người dùng nhập.

Áp dụng mô hình MVC giúp tách biệt tầng Controller – Service – DAO.

Sử dụng JDBC để thao tác dữ liệu với MySQL.

Giao diện người dùng trực quan, sử dụng JavaScript để xử lý các thao tác động.

Contributions:

Thiết kế cấu trúc dự án theo chuẩn MVC, tổ chức lại source code dễ bảo trì.

Xây dựng các module controller, service, DAO.

Viết truy vấn SQL, tối ưu xử lý dữ liệu.

Tích hợp phân trang, validate dữ liệu, xử lý lỗi.

Phát triển toàn bộ giao diện HTML/CSS/JS và binding dữ liệu từ backend.

Kiểm thử, fix bug, hoàn thiện UI/UX.
